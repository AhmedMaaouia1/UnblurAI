# Documentation Images

Ce dossier contient les captures d'écran et images de documentation du projet UnblurAI.

## Images Disponibles

### interface1.png
Capture d'écran de l'interface utilisateur principale montrant l'upload d'image.

### interface2.png
Interface avec sélection de la qualité JPEG et options de restauration.

### interface_resultat.png
Exemple de résultat de restauration avec comparaison avant/après.

## Utilisation

Ces images sont utilisées dans la documentation et le README principal du projet pour illustrer les fonctionnalités et résultats de l'application.

Pour ajouter ces images au README principal :

```markdown
## Interface

![Interface principale](docs/images/interface1.png)

## Résultats

![Avant/Après](docs/images/interface_resultat.png)
```

## Format des Images

- Format : PNG
- Résolution : Variable selon la capture
- Compression : Sans perte (PNG)
